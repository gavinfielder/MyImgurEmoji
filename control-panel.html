Getting Started

  1. Create an Imgur album and add any images you wish. https://imgur.com
  2. Add tag ids in the description of each image, such as :happy:.
  3. Still on the Imgur album, Open the MyImgurEmoji extension and click 'Add This Imgur Album'
  4. You're done! Type @mie :happy:@ in any textarea on any page and watch the results!

Optionally, you can use multiple albums, and/or define how to represent the images on specific websites you visit.

Advanced usage: @mie [-s albumName] :tag_id:@. Omitting -s albumName selects your default album.
